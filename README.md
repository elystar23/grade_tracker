# Grade Tracker Project Proposal

## Project Overview
This is a Python Grade Tracker that lets a user enter grades and calculates their average. It helps students see how they are doing in their classes.

## Use of a List or Collection
The program uses a Python list to store grades. The list lets us save multiple grades and process them together.

Example:
grades = []

## Use of a Function
The program uses a function called `calculate_average(grades)` that takes the list of grades as input and returns the average grade. Changing the grades in the list changes the result.

## Use of a Loop
The program uses a loop to let the user enter multiple grades and to go through the list when calculating the average.

## How to Run the Program
1. Open the project in a Python environment.
2. Run `grade_tracker.py`.
3. Enter your grades when prompted.
4. The program will show your average.
